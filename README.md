# HK3044
midterm and final homework
<br>
1. 原本預計的每日一句<br>
開啟程式後顯示今日日期與今日句子<br>
預計句子格式：<br>
句子<br>
四縣拼音：<br>
海陸拼音：<br>
<br>
要不要做圖片?

2. 查詢功能<br>
可查詢以前的句子(日期格式例：1023)<br>
需判斷輸入是否合法(數字)

3. 測驗功能<br>
分兩個，都可以選擇要考幾題，計算正確率

- 3-1考某段期間的句子<br>
考拼音可能比較簡單(避免字體問題)

- 3-2使用者自訂<br>
使用者自行輸入「詞」與「拼音」，進行測驗。
